# Theodoro De Almeida — Portfolio

A personal portfolio site for a Full-Stack Software Engineer, built with a modern frontend stack.

## Stack
- **Next.js 15** (App Router) + **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (scroll + entrance animations)
- **lucide-react** icons

## Sections
- **Hero** — who I am (full-stack engineer) + quick stats
- **About** — focus areas and skill groups
- **Experience** — most recent role (CodeStream Labs) with project highlights
- **Work** — selected projects with links and imagery
- **Contact** — email + LinkedIn

## Develop
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
npm start        # serve production build
```

## Editing content
All copy lives in [`lib/data.ts`](lib/data.ts) — update the profile, experience, and
projects there without touching the components.
